CipherBox

Encode & decode classic ciphers
Shift:
Caesar cipher shifts each letter by N positions in the alphabet. ROT-3 by default.
A→D, B→E … Z→C (wraps around). Non-letters are unchanged.
ROT-13 is a Caesar cipher with a shift of 13. Applying it twice gives the original text (encode = decode).
Common in online forums to hide spoilers.
Atbash reverses the alphabet: A↔Z, B↔Y, C↔X…
Originally used for Hebrew, it is self-reciprocal — encode and decode are the same operation.
Key:
Vigenère cipher uses a keyword to vary the Caesar shift for each letter. The key repeats to match the message length. Much stronger than simple Caesar.
Morse code reference (word separator = /):
Binary encoding converts each character to its 8-bit ASCII representation.
Example: "A" → 01000001   "a" → 01100001
